## Strict vs Unstrict

Strict Mode has only 2 commands.

* ㅇ : Show "안녕 세계"
* ㅈ : Stop Your program (Ignore next commands) Just use return; command in C

Unstrict Mode has 2 more commands.

* ㅊ : Print message.
 You Can't use english in this language
